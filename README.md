# TD4 CPU on FPGA

## Overview
This project implements a 4-bit TD4 CPU on FPGA to understand the internal architecture of processors.

## Features
- 4-bit CPU
- FPGA implementation
- Instruction execution
- Register and ALU design
- Clock-synchronized operation
